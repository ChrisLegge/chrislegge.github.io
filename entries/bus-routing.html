Predictive Bus Routing

November 2025 – March 2026

Overview

In a team we designed a system that would increase the efficinecy of busses using an machine learning algorithm that uses accurate data modeled from accross the UK to Predict the footfall at each bus stop and dynamically reroute busses to pick up passangers so that you would spend less time waiting and less time travelling.

Physical Board

My main part of the project was to make the physical map, so that people could see at bus stops when the next bus will arrive, I orginally designed a custom PCB for this part of the project. However, this was out of budget for out small project, so instead I soldered the light map together from a long strip of LEDs, then I added machine learning model to my Arduino so that you could see the busses moving on the map. I then repeated this using verilog and and FPGA board so that we could run the different settings and it could run faster.

Results & Reflections

The project is for the Engeneers Without Borders competition, which at the time of writing has not been submitted yet. The project required careful planning from all members of the group so that we could get it done on time, we created a gantt chart to keep on track and assigned tasks to everyones strengths which is why we managed to be so successfull with our project management, overall we worked well as a team together and the project management played to our strengths.
